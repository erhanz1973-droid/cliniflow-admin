# Cliniflow Admin Panel

Clinic admin panel built with Next.js.

## Environment Variables

- `NEXT_PUBLIC_API_URL` - Backend API URL (default: http://localhost:5050)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production

```bash
npm run build
npm start
```

## Render Deployment

Set `NEXT_PUBLIC_API_URL` to your backend API URL in Render environment variables.

Build Command: `npm run build`
Start Command: `npm start`
